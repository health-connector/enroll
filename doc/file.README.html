Enroll

Enroll App is the web-facing, transactional component of the first built-to-purpose, open source eligibility and enrollment solution for Health Benefit Exchanges. Enroll powers the DC Health Link site for the DC Health Benefit Exchange, and is developed and managed by IdeaCrew and HBX.
